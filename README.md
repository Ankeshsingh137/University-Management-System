# 🎓 University Management System

A desktop-based **University Management System** developed using **Java (AWT & Swing)** and **MySQL**. This application helps manage student, faculty, and university-related operations efficiently.

---

## 🚀 Features

- 🔐 User Login & Authentication  
- 👨‍🎓 Add Student Details  
- 📋 View Student Information  
- 🧑‍🏫 Manage Faculty Details  
- 📚 Course Management  
- 💰 Fee Management  
- 🧾 Record Management System  
- 📊 Easy Navigation Dashboard  

---

## 🛠️ Technologies Used

- Java (AWT & Swing)  
- MySQL Database  
- JDBC (Java Database Connectivity)  
- NetBeans IDE
